#DATA_SYNTHESIS #FEATURE_LIFECYCLE

MaaS: User Feedback & Churn Analysis Feature

The Challenge

Addressing information gaps in a Mobility-as-a-Service (MaaS) platform. Faced a systemic blind spot where monthly churn lacked qualitative feedback, requiring a data-driven strategy to identify root causes and define retention features.

Key Execution & Tools

  • Analysis: Used SQL (BigQuery) to extract data, synthesizing results into Excel/PPT reports to provide actionable insights for service PMs.
  • Planning: Owned the feature lifecycle, coordinating with developers and designers via LeSS to ensure PRD/PBR documentation alignment.
  • Issue Triage: Utilized Datadog for error monitoring, distinguishing between UX friction and technical bugs to streamline Jira tickets.
#LOGIC_MAPPING #UAT_METHODOLOGY

SCM: Store Inventory & Asset Management System

The Challenge

Navigating critical information silos in a global supply chain project. Bridged the gap between regional technical requirements (SG/SZ) and local operational realities in TW/PH markets.

Key Execution & Tools

  • Alignment: Used Miro and Confluence to map complex inventory flows (transfers, stocktakes) for cross-border alignment.
  • Validation: Led UAT sessions, translating operational pain points into structured system test cases to ensure logic alignment for the P0 launch.
  • Documentation: Developed 5 training materials and tracked system logic using Excel to ensure high adoption rates post-launch.
